Flexbox Grid Sass
===========

Sass Version of FlexboxGrid 
[flexboxgrid.com](http://flexboxgrid.com)

Development Install
---------
### npm
```
git clone https://github.com/mdtanrikulu/flexboxgrid.git
cd flexbox
npm install
```

### compile & minify

```
npm run compile
```

Then you can find minified and/or normal css file in the dist folder.
